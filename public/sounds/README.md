# Sounds

Placeholder audio generated locally via `python3 wave` stdlib (Phase A). Short synthesized tones (~1 s each) that stand in for the XP boot ceremony sounds.

- `login.wav` — two-note rising chime, plays when user clicks the login avatar.
- `logoff.wav` — two-note falling chime, reserved for Phase C Turn Off dialog.
- `balloon.wav` — short pling, plays with the welcome balloon ~2 s after login.

**Pixel polish follow-up:** replace with public-domain XP sounds from archive.org (or better-curated alternatives) once network access is available. These files are drop-in replaceable — same filenames, any short 16-bit PCM WAV works.
